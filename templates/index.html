Squash me before pull request!

This is a simple tool to allow having GitHub pull requests marked as success or failure based on whether commits have been squashed or not. The idea is to only merge fully squashed PRs to keep the project log nice and clean.

Want to give it a try?

  • Get commits

    In order to squash your last three commits in one and type following command

    git rebase -i HEAD~3

    This will open up your editor with the following

  • Squash it

    Change your file to this:

    pick f392171 Added new feature X squash ba9dd9a Added new elements to page design squash df71a27 Updated CSS for new elements

    When done, save and quit your editor. Git will now squash the commits into one. All done!

  • Enjoy

    Now you can relax, open bottle of wine and enjoy that your history of commits look pretty.
