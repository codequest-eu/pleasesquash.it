This is a simple tool to allow having GitHub pull requests marked as success or failure based on whether commits have been squashed or not. The idea is to only merge fully squashed PRs to keep the project log nice and clean.

In order to achieve that you need to provide your full GitHub repo name - eg. codequest-eu/pleasesquash.me and grant the tool access to your GitHub statuses. This will not grant access to your source code. You will be guided through the rest of the setup (which involves one extra step) after submitting the form below.
